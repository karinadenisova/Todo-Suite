# ✅ Todo Suite — 7 языков, один планировщик задач

Коллекция из семи независимых реализаций **To-Do листа** с расширенным функционалом. Каждая версия включает:

- ➕ Добавление задач (название, описание, приоритет, дедлайн)
- ✅ Отметка выполнения / восстановление
- ✏️ Редактирование любой задачи
- 🗑️ Удаление задач
- 🎚️ Приоритеты: Высокий, Средний, Низкий
- 📅 Дедлайны (дата + время, где поддерживается)
- 🔍 Поиск по названию и описанию
- 🧹 Фильтрация: все / активные / выполненные / по приоритету / по дате
- 💾 Автосохранение в файл `tasks.json` (или localStorage для JS)
- 📊 Красивое табличное отображение (цветные метки в CLI)

## Языки и технологии

| Язык      | Интерфейс      | Хранение            | Фишки                                                                 |
|-----------|----------------|----------------------|-----------------------------------------------------------------------|
| Python    | CLI (colorama)  | JSON                 | Цветные приоритеты, ввод дат, динамическое меню                       |
| JavaScript| Веб (HTML/CSS)  | localStorage         | Drag & drop, фильтры в реальном времени, экспорт/импорт CSV           |
| Go        | CLI (bubbletea) | JSON                 | TUI с виджетами (альтернативно – простое меню)                        |
| Rust      | CLI (dialoguer) | JSON (serde)         | Автодополнение, цветной вывод, обработка ошибок                       |
| Java      | CLI (Scanner)   | JSON (Jackson)       | Потоковая безопасность, валидация дат                                 |
| C#        | CLI (Spectre.Console) | JSON (System.Text.Json) | Таблицы с иконками, календарный выбор даты                           |
| PHP       | Веб (Bootstrap) | JSON-файл            | Кнопки действий, сортировка, адаптивный дизайн                        |
